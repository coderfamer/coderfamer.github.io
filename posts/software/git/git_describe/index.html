git describe 命令详解

git 获取最新的 tag

查找最近的tag

git describe
# 不需要后缀
git describe --abbrev=0

获取当前分支的tag

git describe --abbrev=0 --tags

获取所有分支的tag

git describe --tags `git rev-list --tags --max-count=1`
